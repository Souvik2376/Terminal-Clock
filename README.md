# Terminal Clock

This program is created using Shell Script . The clock Shows the live time with a fantastic design .


## Run Locally

Clone the project

```bash
  git clone https://github.com/Souvik2376/Terminal-Clock.git
```

Go to the project directory

```bash
  cd Terminal Clock
```

Give Permission

```bash
  chmod 777 clock.sh
```

Run

```bash
  ./clock.sh
```
